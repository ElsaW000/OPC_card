# OPC 名片小程序 - 页面交互框架 (v2.0)

> Last Updated: 2026-03-25
> 此版本为最新确定版

## 页面结构

### TabBar 导航（4个）
- 我的名片 (mycards) - 首页，交换名片入口
- 联系人 (contacts) - 交换过的联系人
- 工作台 (home) - 功能入口
- 管理 (management) - 设置

### 独立页面
| 页面 | 路径 | 说明 |
|------|------|------|
| 交换名片 | pages/exchange/exchange | 扫码交换流程 |
| 名片详情 | pages/cardDetail/cardDetail | 他人看到的我的名片/预览 |
| 编辑名片 | pages/edit/edit | 编辑名片 |
| 访客记录 | pages/visitor/visitor | 谁看了我的名片 |
| 数据分析 | pages/analytics/analytics | 数据统计 |
| 会员权益 | pages/member/member | PRO会员 |
| AI功能 | pages/aiFeatures/aiFeatures | AI助手 |

## 页面跳转关系

### 1. 我的名片 (mycards) - TabBar首页
- 底部有蓝色"交换名片"按钮 → 跳转exchange页面扫码
- 每个名片可以当面扫码交换
- 点击卡片 → 预览自己的名片详情（cardDetail，有编辑/分享按钮）
- 点击"编辑" → 跳转到edit页面
- 点击"分享" → 分享到微信/小红书等其他平台
- 点击"设为默认" → 更新默认名片
- 点击"+" → 创建新身份

### 2. 联系人 (contacts) - TabBar
- 这是**交换过名片**的联系人列表
- 点击联系人 → 查看联系人名片详情（cardDetail）
- 点击"聊天" → 跳转到微信（触发微信聊天）
- 点击"电话" → 弹出确认拨打对话框
- 点击"备忘录" → 添加/查看联系人备忘录

### 3. 工作台 (home) - TabBar
**a. AI社交助理**
- 对话框形式
- 默认招呼语："您有什么需求？"、"看下最近圈子里有什么新鲜事？"
- AI助手分析联系人和访客

**b. 数据模块**
- 访客数据：每周总访问数字
- 访客画像：气泡图（气泡大小=数量，标签=访客类型）
- 点击箭头 → 跳转到数据分析页面 (analytics)

**c. 星标联系人**
- 跳转联系人页面，筛选展示星标联系人
- 点击联系人 → 查看名片详情
- 当联系人更新产品/视频时，显示红点提示

**d. 系统设置**
- AI语气设置
- 是否公开名片动态
- 隐私设置

**e. 底部**
- 会员权益介绍
- 联系我们 → 弹出表单收集建议

### 4. 访客记录 (visitor)
- 展示访客列表（谁扫了我的二维码）
- 免费用户：只显示前3个基础信息（无联系方式）
- 付费用户：显示完整信息+名片详情

### 5. 数据分析 (analytics)
- 访客数据详情
- 交换数据统计
- AI优化建议

### 6. AI功能 (aiFeatures)
- AI悬浮球（右侧悬浮，可展开/收起）
- 来访者分析
- 联系人推荐
- 话术建议

### 7. 管理 (management) - TabBar
- 账号设置
- 名片模板管理

## 名片详情页 (cardDetail) - 两种视图

### 视图A：他人看到的（他人扫二维码）
- 完整展示：无编辑按钮
- 只有一个"交换名片"按钮
- 点击"交换名片" → 触发交换流程

### 视图B：自己预览的（从mycards点击卡片）
- 完整展示：有编辑和分享按钮
- 点击"编辑" → 跳转到edit页面
- 点击"分享" → 分享到微信/小红书等

---

## 交换名片入口

1. **我的名片页面底部** - 蓝色"交换名片"大按钮 → 跳转exchange扫码
2. **我的名片每个名片** - 可当面扫码交换
3. **分享** - mycards/cardDetail分享到微信对话、小红书等

---

## 技术可行性确认

1. **分享到小红书**：微信小程序支持直接分享到微信，需要确认是否支持分享到小红书（可能需要App拉起）
2. **跳转到微信**：使用wx.openCustomerServiceChat或weixin://打开微信
3. **电话拨打**：wx.makePhoneCall

---

## 访客记录逻辑

- 免费用户：只展示前3个访客的基础信息（姓名+头像），不含联系方式
- 付费用户：展示所有访客的完整信息，包含联系方式、产品介绍、作品集、视频等
