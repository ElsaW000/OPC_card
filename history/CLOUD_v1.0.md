# OPC 名片小程序 - 云开发配置指南

## 1. 开通云开发

1. 打开微信开发者工具
2. 点击右上角「云开发」按钮
3. 按提示开通云开发
4. 获取环境 ID（会生成一个类似 `opc-card-xxx` 的环境 ID）

## 2. 配置环境 ID

打开 `app.js`，把 `env` 替换为你的环境 ID：

```javascript
wx.cloud.init({
  env: 'your-cloud-env-id', // 替换这里
  traceUser: true
})
```

## 3. 创建数据库集合

在微信开发者工具的「云开发控制台」→ 「数据库」中创建以下集合：

- `cards` - 存储名片数据

## 4. 上传云函数

1. 在微信开发者工具中，右键点击 `cloudfunctions/getCard` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 同样操作 `cloudfunctions/saveCard` 文件夹

## 5. 完成

配置完成后，小程序会自动从云端读取/保存名片数据。

---

## 云函数说明

| 云函数 | 作用 |
|--------|------|
| getCard | 获取用户名片（自动创建默认名片） |
| saveCard | 保存用户名片 |

## 本地测试

如果不使用云开发，页面会使用本地模拟数据（在 index.js 和 edit.js 中）。
