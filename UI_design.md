# OPC 名片小程序 - UI 设计文档

> 本文档定义了 OPC 名片小程序的页面结构、交互设计和视觉规范。

---

## 1. 页面结构

### TabBar 配置（4个Tab + 1个重要按钮）

| Tab | 页面路径 | 标题 |
|-----|----------|------|
| 1 | pages/mycards/mycards | 我的名片 |
| 2 | pages/contacts/contacts | 联系人 |
| 3 | (中间蓝色圆按钮) | 交换名片 |
| 4 | pages/home/home | 工作台 |
| 5 | pages/management/management | 管理 |

**特殊设计**：TabBar 中间有一个突出的蓝色圆形"交换名片"按钮 🎯

### 页面清单

```
pages/
├── mycards/           # 我的名片（Tab 1）
│   └── cardDetail/   # 名片展示页（从我的名片跳转）
├── contacts/          # 联系人（Tab 2）
├── home/              # 工作台（Tab 4）
│   ├── visitor/       # 访客（从工作台跳转）
│   ├── analytics/     # 数据分析（从工作台跳转）
│   ├── member/        # 会员权益（从工作台跳转）
│   └── aiFeatures/    # AI功能（从工作台跳转）
├── management/        # 管理（Tab 5）
│   └── qrcode/        # 名片二维码（从管理跳转）
├── visitor/           # 访客（独立页面）
├── analytics/         # 数据分析（独立页面）
├── member/            # 会员权益（独立页面）
├── aiFeatures/        # AI功能（独立页面）
├── edit/              # 编辑名片
└── webview/           # 网页视图
```

**不在TabBar的页面**：
- 交换记录（从联系人跳转）
- 交换名片（中间蓝色按钮触发）
- 名片展示页（从我的名片跳转）
- 访客、数据、会员权益、AI功能（从工作台跳转）

## 2. 页面详细设计

### 2.1 我的名片页 (mycards) ⭐ Tab 1

**位置**：Tab 1

**标题**：
- 主标题："我的名片"
- 副标题："MULTI-IDENTITY MANAGEMENT"

**布局**：
- 垂直滚动卡片流
- 每张名片是独立的圆角矩形大卡片

**卡片示例**：

**第一张（技术身份）**
- 背景：亮蓝色渐变
- 内容：姓名"陈小独立"、职位"Full-stack Developer"、公司"CODEFLOW AI STUDIO"
- 右上角："默认"标签 + 二维码图标
- 右侧：方形圆角头像

**第二张（商务身份）**
- 背景：深灰蓝色调
- 内容：姓名"Independent Chen"、职位"Founder & CEO"、公司"ONE PERSON COMPANY LTD."

**第三张（个人社交）**
- 背景：翠绿色渐变（露出顶部）

**功能按钮**（每张名片下方）：
- 设为默认 / 当前默认
- 编辑
- 分享
- 删除

**交互**：
- 点击卡片 → 跳转到名片展示页 (cardDetail)
- 右上角二维码图标 → 跳转到名片二维码页

**添加按钮**：
- 右下角：蓝色圆形"+"号

## 3. 视觉规范

### 3.1 色彩

| 用途 | 颜色 |
|------|------|
| 主色（选中） | #2563EB (蓝色) |
| 背景 | #F8F9FA |
| 卡片背景 | #FFFFFF |
| 文字主色 | #000000 |
| 文字次色 | #666666 |
| 文字辅助 | #999999 |
| 成功 | #10B981 |
| 警告 | #F59E0B |
| 错误 | #EF4444 |

### 3.2 名片渐变

| 类型 | 渐变 |
|------|------|

| 商务身份 | 深灰蓝色调 |

### 3.3 间距

- 页面边距：32rpx
- 卡片内边距：24rpx
- 元素间距：16rpx

### 3.4 圆角

- 大卡片：32rpx
- 中卡片：24rpx
- 小元素：16rpx

### 3.5 阴影

- 卡片阴影：0 8rpx 24rpx rgba(0,0,0,0.06)
- 浮动按钮：0 16rpx 48rpx rgba(47,101,238,0.3)

## 4. 交互逻辑

### 4.1 交换流程

**方式一：中间蓝色按钮**
```
点击TabBar中间蓝色圆形按钮
    → 交换名片页面
    → 扫码或搜索对方
    → 确认交换
    → 添加到联系人
```

**方式二：从名片展示页**
```
点击"分享名片"
    → 选择交换方式
    → 完成交换
```

### 4.2 导航逻辑

- Tab 切换：4个主Tab
- 交换名片：中间蓝色按钮触发（全屏页面）
- 访客/数据/会员/AI：从工作台跳转

### 4.3 访客来源追踪

| 来源 | 追踪方式 |
|------|----------|
| 扫码 | 扫描小程序码 |
| 链接分享 | 带参数的分享链接 |
| 微信搜索 | 微信搜索入口 |
| 名片转发 | 名片转发记录 |

### 4.4 会员权限

| 功能 | 免费版 | PRO版 |
|------|--------|-------|
| 访客查看 | 仅3条 | 无限 |
| 来源分析 | 基础 | 详细 |
| AI功能 | 有限次数 | 无限 |
| 名片数量 | 3张 | 无限 |

---

## 5. 设计风格

- **现代简约** (Modern Minimalism)
- 大圆角设计
- 微渐变色值
- 细腻投影（Soft Shadows）
- 背景：浅灰色或白色
- TabBar 中间突出按钮是视觉焦点
