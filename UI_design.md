# OPC 名片小程序 - UI 设计文档

> 本文档定义了 OPC 名片小程序的页面结构、交互设计和视觉规范。

---

## 1. 页面结构

### TabBar 配置（4个Tab）

| Tab | 页面路径 | 标题 |
|-----|----------|------|
| 1 | pages/mycards/mycards | 我的名片 |
| 2 | pages/contacts/contacts | 联系人 |
| 3 | pages/home/home | 工作台 |
| 4 | pages/management/management | 管理 |

**注意**：数据和访客不是独立Tab，而是从工作台跳转的子页面。

### 页面清单

```
pages/
├── mycards/        # 我的名片（Tab 1）
├── contacts/       # 联系人（Tab 2）
├── home/           # 工作台（Tab 3）
├── management/     # 管理（Tab 4）
├── visitor/        # 访客（从工作台跳转）
├── analytics/      # 数据分析（从工作台跳转）
├── edit/           # 编辑名片
└── webview/       # 网页视图
```

## 2. 页面详细设计

### 2.1 我的名片页 (mycards) ⭐ Tab 1

**位置**：Tab 1（默认选中）

**标题**：
- 主标题："我的名片"
- 副标题："MULTI-IDENTITY MANAGEMENT"

**布局**：
- 垂直滚动卡片流
- 每张名片是独立的圆角矩形大卡片

**卡片示例**：

**第一张（技术身份）**
- 背景：亮蓝色渐变
- 内容：姓名"陈小独立"、职位"Full-stack Developer"、公司"CODEFLOW AI STUDIO"
- 右上角："默认"标签
- 右侧：方形圆角头像

**第二张（商务身份）**
- 背景：深灰蓝色调
- 内容：姓名"Independent Chen"、职位"Founder & CEO"、公司"ONE PERSON COMPANY LTD."

**第三张（个人社交）**
- 背景：翠绿色渐变（露出顶部）
- 内容：个人社交信息

**功能按钮**（每张名片下方）：
- 设为默认 / 当前默认
- 编辑
- 分享
- 删除

**添加按钮**：
- 右下角：蓝色圆形"+"号

---

### 2.2 联系人页 (contacts) ⭐ Tab 2

**位置**：Tab 2

**结构**：
```
├── 搜索栏
│
├── 标签筛选（Pills）
│   ├── 全部（默认选中，蓝色高亮）
│   ├── AI
│   ├── B端产品
│   ├── 出海
│   ├── Flutter
│   └── ...（可扩展）
│
├── 星标联系人（顶部）
│
└── 联系人列表
    └── 卡片（带微阴影的白色圆角矩形）
        ├── 左侧：正方形圆角头像（可带金色星星标识）
        ├── 中间：
        │   ├── 名字（加粗） + 活跃时间（10分钟前、2小时前）
        │   ├── 公文包图标 + 职位 @ 公司
        │   └── 技能/领域标签组
        └── 底部：
            ├── 蓝色边框"聊天"按钮
            └── 电话图标按钮
```

### 2.3 工作台 (home) ⭐ Tab 3

**位置**：Tab 3

**结构**：
```
├── 顶部：大卡片（当前默认名片预览）
```

## 3. 视觉规范

### 3.1 色彩

| 用途 | 颜色 |
|------|------|
| 主色（选中） | #2563EB (蓝色) |
| 背景 | #F8F9FA |
| 卡片背景 | #FFFFFF |
| 文字主色 | #000000 |
| 文字次色 | #666666 |
| 文字辅助 | #999999 |
| 成功 | #10B981 |
| 警告 | #F59E0B |
| 错误 | #EF4444 |

### 3.2 名片渐变

| 类型 | 渐变 |
|------|------|
| 技术身份 | 亮蓝色渐变 |
| 商务身份 | 深灰蓝色调 |
| 个人社交 | 翠绿色渐变 |

### 3.3 间距

- 页面边距：32rpx
- 卡片内边距：24rpx
- 元素间距：16rpx

### 3.4 圆角

- 大卡片：32rpx
- 中卡片：24rpx
- 小元素：16rpx

### 3.5 阴影

- 卡片阴影：0 8rpx 24rpx rgba(0,0,0,0.06)
- 浮动按钮：0 16rpx 48rpx rgba(47,101,238,0.3)

## 4. 交互逻辑

### 4.1 交换流程

```
用户点击"交换名片" 
    → 交换确认页（预览对方名片）
    → 确认交换 
    → 添加到联系人列表
```

### 4.2 导航逻辑

- Tab 切换：4个主Tab
- 子页面：从工作台跳转到访客/数据分析

### 4.3 会员权限

- 访客详情查看：PRO 会员
- 最近访客仅显示3条：非会员

---

## 5. 设计风格

- **现代简约** (Modern Minimalism)
- 大圆角设计
- 微渐变色值
- 细腻投影（Soft Shadows）
- 背景：浅灰色或白色
