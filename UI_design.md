# OPC 名片小程序 - UI 设计文档

> 本文档定义了 OPC 名片小程序的页面结构、交互设计和视觉规范。

---

## 1. 页面结构

### TabBar 配置（5个Tab）

| Tab | 页面路径 | 标题 | 图标 |
|-----|----------|------|------|
| 1 | pages/home/home | 工作台 | grid |
| 2 | pages/mycards/mycards | 名片 | card |
| 3 | pages/contacts/contacts | 联系人 | user |
| 4 | pages/visitor/visitor | 访客 | eye |
| 5 | pages/analytics/analytics | 数据 | chart |

### 页面清单

```
pages/
├── home/           # 工作台
├── mycards/        # 名片（我的名片）
├── contacts/       # 联系人
├── visitor/        # 访客
├── analytics/      # 数据分析
├── edit/           # 编辑名片
└── webview/        # 网页视图
```

## 2. 页面详细设计

### 2.1 工作台 (home)

**位置**：Tab 1

**结构**：
```
├── 顶部：大卡片（当前默认名片预览）
│   ├── 背景：模糊的 banner 图
│   ├── 左侧：名字、职位、公司
│   └── 右侧：方形圆角头像
│
├── 快捷入口：2x2 宫格
│   ├── 名片管理 → 跳转 mycards tab
│   ├── 浏览记录 → 跳转 visitor tab
│   ├── 交换记录 → 跳转 contacts tab
│   └── 数据分析 → 跳转 analytics tab
│
├── 最近访客（仅显示3个）
│   └── 卡片列表
│
└── 底部：浮动"交换名片"按钮
```

---

### 2.2 名片页 (mycards)

**位置**：Tab 2

**结构**：
```
├── 顶部标题
│   ├── 主标题："我的名片"
│   └── 副标题："MULTI-IDENTITY MANAGEMENT"
│
├── 切换指示器（横向滚动）
│   ├── 1 / 2 / 3
│   └── + （添加）
│
├── 当前名片预览（大卡片）
│   ├── 渐变背景（按身份类型）
│   ├── 名字、职位、公司
│   ├── "默认"标签（如果默认）
│   └── 方形圆角头像
│
├── 功能按钮
│   ├── 编辑 → edit 页面
│   ├── 分享
│   └── 设为默认 / 当前默认
│
├── 名片列表（所有名片）
│   └── 点击切换
│
└── 底部：蓝色圆形"+"添加按钮
```

**名片类型样式**：
- 技术身份：亮蓝色渐变
- 商务身份：深灰蓝色调
- 个人社交：翠绿色渐变

## 3. 视觉规范

### 3.1 色彩

| 用途 | 颜色 |
|------|------|
| 主色（选中） | #2563EB (蓝色) |
| 次色 | #2F65EE |
| 背景 | #F8F9FA |
| 卡片背景 | #FFFFFF |
| 文字主色 | #000000 |
| 文字次色 | #666666 |
| 文字辅助 | #999999 |
| 成功 | #10B981 |
| 警告 | #F59E0B |
| 错误 | #EF4444 |

### 3.2 间距

- 页面边距：32rpx
- 卡片内边距：24rpx
- 元素间距：16rpx

### 3.3 圆角

- 大卡片：32rpx
- 中卡片：24rpx
- 小元素：16rpx

### 3.4 阴影

- 卡片阴影：0 8rpx 24rpx rgba(0,0,0,0.06)
- 浮动按钮：0 16rpx 48rpx rgba(47,101,238,0.3)

## 4. 交互逻辑

### 4.1 交换流程

```
用户点击"交换名片" 
    → 交换确认页（预览对方名片）
    → 确认交换 
    → 添加到联系人列表
```

### 4.2 浏览记录

- 谁看过我：访客页
- 我浏览了谁：需要记录用户主动查看的名片

### 4.3 会员权限

- 访客详情查看：PRO 会员
- 最近访客仅显示3条：非会员
