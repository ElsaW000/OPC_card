# OPC 名片 - 设计规范

## 一、设计理念
- **简洁专业**：适合商务场景
- **现代质感**：渐变、毛玻璃、大圆角
- **清晰层级**：信息分组明确

---

## 二、色彩系统

### 主色
| 颜色 | 色值 | 用途 |
|------|------|------|
| 主蓝 | #2563EB | 按钮、激活态、链接 |
| 深蓝 | #1D4ED8 | 强调、次级按钮 |
| 浅蓝 | #EFF6FF | 背景、选中底色 |

### 中性色
| 颜色 | 色值 | 用途 |
|------|------|------|
| 深黑 | #111827 | 标题、重要文字 |
| 灰 | #64748B | 正文、次要文字 |
| 浅灰 | #94A3B8 | 提示、占位符 |
| 边框 | #F1F5F9 | 分割线、边框 |

### 渐变色
- 蓝渐变：`linear-gradient(135deg, #3B82F6 0%, #1D4ED8 100%)`
- 绿渐变：`linear-gradient(135deg, #10B981 0%, #059669 100%)`
- 紫渐变：`linear-gradient(135deg, #8B5CF6 0%, #A855F7 100%)`

## 七、页面结构规范

### 通用页面结构
```
┌─────────────────────┐
│     Header (56rpx)  │  ← 固定顶部，白色背景
├─────────────────────┤
│                     │
│     Content         │  ← 可滚动区域
│     (flex: 1)       │
│                     │
├─────────────────────┤
│   TabBar (100rpx)   │  ← 固定底部
└─────────────────────┘
```

### 页面层级
1. Header：固定定位，z-index: 40
2. Content：flex: 1，可滚动
3. TabBar：固定定位，z-index: 999

## 八、核心组件规范

### 1. 按钮
- 主按钮：深色渐变，圆角 full，高度 88rpx
- 次按钮：白色背景 + 边框，高度 80rpx
- 图标按钮：正方形 80rpx，圆角 50%

### 2. 卡片
- 白色背景
- 圆角 24rpx
- 内边距 24rpx
- 阴影

### 3. 输入框
- 高度 80rpx
- 圆角 16rpx
- 边框 2rpx
- focus 状态：边框变蓝

### 4. 列表项
- 高度 最小 88rpx
- 左边距 24rpx
- 底部分割线
