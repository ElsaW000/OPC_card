# OPC 名片 - 竞品分析与付费功能规划

## 一、竞品分析

### 1.1 名片全能王

| 指标 | 数据 |
|------|------|
| 用户数 | 7亿 |
| 收入占比 | 85%来自3款APP |
| 盈利状态 | AI商业化已跑通 |

**他们的功能**：
- AI自动识别名片
- 高级模板
- 企业版/团队管理
- CRM客户管理
- 数据分析（谁看了名片）

**我们不做**：
- 复杂的企业管理
- 重度CRM

---

### 1.2 我们的差异化定位

| 对手 | 我们 |
|------|------|
| 商务正式 | 年轻社交 |
| 功能复杂 | 极简 |
| 收费贵 | 免费/低价 |
| 交换联系方式 | 交换个人上下文 |

**核心区别**：传统名片是"交换联系方式"，我们是"交换个人上下文"——展示我是谁、我在做什么、我能和你如何连接。

---

### 1.3 目标用户

面向 AI 时代快速涌现的新个体与轻团队：
- OPC / 一人公司
- 独立开发者
- 小公司或微型团队创始人
- 独立游戏制作人
- 漫剧作者 / 创作者
- 设计师 / 插画师 / 数字内容创作者
- 项目制自由职业者

他们不是传统老板或销售，而是"交换个人上下文的人"——发出去的是"我最近在做什么项目、我是什么风格的人、我能提供什么能力"。

---

## 二、我们的核心功能（免费）

### 2.1 基础功能

- 名片展示（头像、姓名、职位、公司）
- 联系方式（电话、邮箱）
- 复制/拨打
- 二维码分享
- 基础模板

### 2.2 AI快速生成

- **拍照名片** → 自动提取信息
- **截图公司官网** → 自动抓取内容
- **链接上传** → 自动生成

### 2.3 社交属性

- **碰一碰** - 碰一碰打开名片 → 小程序内确认交换
- **远程发名片** - 微信聊天中转发小程序名片，对方点击可查看并回传
- **小红书一键分享** - 生成分享图/链接

---

## 三、付费功能规划

### 3.1 免费版

- 1个名片
- 基础模板
- 基础分享

### 3.2 付费版（高级功能）

| 功能 | 说明 | 定价参考 |
|------|------|----------|
| AI信息提取 | 拍照/截图自动生成名片 | 付费 |
| 更多模板 | 设计师模板、节日主题 | 付费 |
| 产品/项目展示 | 多个产品展示位 | 付费 |
| 自定义域名 | 专属链接 like jie.me | 付费 |
| 数据分析 | 谁看了你的名片 | 付费 |
| 企业版 | 团队管理、多人名片 | 订阅制 |

### 3.3 变现模式

1. ** Freemium ** - 基础免费，高级功能付费
2. ** 模板付费 ** - 单次付费/打包
3. ** 会员订阅 ** - 月费/年费
4. ** 定制服务 ** - 帮企业定制
