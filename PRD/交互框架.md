# OPC 名片小程序 - 页面交互框架

## 页面结构

```
TabBar 导航
├── 首页 (index) - 交换名片入口
├── 联系人 (contacts) - 我的联系人
├── 工作台 (home) - 功能入口
└── 我的 (my) - 个人中心

独立页面
├── 我的名片 (mycards) - 名片管理
├── 名片详情 (cardDetail) - 他人看到的我的名片
├── 编辑名片 (edit) - 编辑名片
└── 访客记录 (visitor)
```

## 页面跳转关系

### 1. 首页 (index)
- 点击"交换名片" → 扫描二维码 → 获取对方名片 → 添加到联系人

### 2. 联系人 (contacts)
- 点击联系人 → 跳转到该联系人的**名片详情页** (cardDetail)
- 点击"编辑" → 跳转到编辑页 (edit)

### 3. 我的名片 (mycards)
- 点击卡片 → 跳转到名片详情页 (cardDetail)
- 点击"编辑" → 跳转到编辑页 (edit)
- 点击"分享" → 触发分享
- 点击"设为默认" → 更新默认名片

### 4. 名片详情页 (cardDetail)
- **这是他人扫二维码看到的页面**
- 包含：Banner + 头像 + 个人信息 + 作品集 + 视频
- 点击"交换名片" → 交换名片

### 5. 编辑名片 (edit)
- 保存 → 返回上一页
- 取消 → 返回

### 6. 工作台 (home)
- 点击各个模块 → 跳转到对应功能

## 关键逻辑

1. **交换名片入口** ：
   - 全屏悬浮按钮，点击跳转名片详情页，确认要交换的名片，点击"交换名片"
   - mycards和工作台顶部名片，用户可以通过扫描二维码交换名片
   - 访客页面，用户可以在访客页面点击"交换名片"，交换名片到联系人（付费功能）
   - mycards和名片详情页，用户可以在名片详情页点击"分享"，分享名片到微信好友和其他社交媒体平台

1. **名片详情页 (cardDetail)** 视图：
   - 他人看到的样子：完整展示，无编辑功能，属于preview，从mycards点击名片预览
   - 名片编辑完成：用户可以在编辑页保存名片，跳转名片详情页，展示最新的名片信息
   - 入口：a.交换名片时预览入口 b.编辑名片完成后预览入口 c.分享名片预览入口 d.名片管理点击名片预览

2. **默认名片**：用户可以设置多张名片中的一张为默认，用于交换和分享，默认名片在首页和工作台突出展示

3. **访客记录**：展示谁浏览过我的名片，从工作台进入访客记录页面, 可以查看访客的姓名、头像、手机号、邮箱等信息;免费用户展示前三的基础信息个人介绍不含联系方式，付费完整展示，包含所有访客名片详情信息（联系方式，产品介绍，作品集，视频等）

4. **数据分析**：通过工作台入口，展示名片的浏览和交换数据，以及AI优化建议
5. **管理设置**：通过工作台入口，提供名片模板管理、账号设置等功能
6. **跟进任务**：付费功能：AI助手功能新增核心模块，展示待跟进联系人（2-5个），推荐动作：加微信、发消息、二次触达等，按照客户输入的需求推荐最合适的动作。比如最近需要找什么业务方向或者什么行业，AI秘书自动分析联系人及访客进行推荐。
