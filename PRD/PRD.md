# OPC 名片小程序 - 产品需求文档 (v2.0)

> Last Updated: 2026-03-25
> Version 2.0 - 整合版

---

## 一、产品定位

帮助 AI 时代的新个体，更轻、更快地介绍自己、建立连接、沉淀联系人。

**它不是传统销售型商务名片，而是更适合展示"我是谁、我在做什么、我能和你如何连接"的个人连接工具。**

---

### 产品策略
- **微信小程序**：获客渠道，用于快速传播和获取新用户
- **APP**：最终产品，用户注册后会导流到APP使用完整功能

---

## 二、目标用户

面向 AI 时代快速涌现的新个体与轻团队：
- OPC / 一人公司
- 独立开发者
- 小公司或微型团队创始人
- 独立游戏制作人
- 漫剧作者 / 创作者
- 设计师 / 插画师 / 数字内容创作者
- 项目制自由职业者

## 三、字段设计

### 按功能分类

#### 1. 基础信息
- 姓名
- 头像
- 身份标签
- 一句话介绍
- 学校/学历（可选：学信网绑定验证 - 后期功能，需要API对接）

#### 2. 联系方式（可设置公开/非公开）
- 电话
- 邮箱
- 微信
- 其他联系方式

#### 3. 职业信息
- 职位
- 公司
- 当前项目
- 工作年限

#### 4. 作品/产品展示
- 作品链接
- 项目展示
- 视频

#### 5. 社交媒体
- GitHub
- Twitter/X
- 抖音/B站
- 其他社媒入口

#### 6. 隐私设置（安全性）
- **公开**：拿到名片的人都能看到

- 每个字段可单独设置

## 五、页面跳转关系

### 5.1 我的名片 (mycards) - TabBar首页
- 底部有蓝色"交换名片"按钮 → 跳转exchange页面扫码
- 每个名片可以当面扫码交换
- 点击卡片 → 预览自己的名片详情（cardDetail，有编辑/分享按钮）
- 点击"编辑" → 跳转到edit页面
- 点击"分享" → 分享到微信/小红书等其他平台
- 点击"设为默认" → 更新默认名片
- 点击"+" → 创建新身份

### 5.2 联系人 (contacts) - TabBar
- 这是**交换过名片**的联系人列表
- **待处理请求** - 顶部显示有人要跟你交换名片
  - 点同意 → 交换成功，双方成为联系人
  - 点拒绝 → 拒绝交换
- **AI助手提示**：顶部显示"谁谁谁更新了产品哦，可以去看看"（如果有更新）
- 点击联系人 → 查看联系人名片详情（cardDetail）
- 点击"电话" → 弹出确认拨打对话框 (wx.makePhoneCall)
- 点击"备忘录" → 添加/查看联系人备忘录
- （"聊天"按钮暂时不实现，如无法跳转微信则去掉）

### 5.3 工作台 (home) - TabBar
**a. AI社交助理**
- 对话框形式，和AI功能页面(aiFeatures)是同一个
- 客户手动输入一句话 → 跳转到AI功能页面继续对话
- 默认招呼语："您有什么需求？"、"看下最近圈子里有什么新鲜事？"
- AI助手分析联系人和访客，可选模型（免费额度有限，部分付费）

**b. 数据模块**
- 访客数据：每周总访问数字
- 访客画像：气泡图/云图（按名片编辑时的AI识别标签分类，气泡大小=数量）
- 点击箭头 → 跳转到数据分析页面 (analytics)

**c. 星标联系人**
- 跳转联系人页面，筛选展示星标联系人
- 点击联系人 → 查看名片详情
- 当联系人更新产品/视频时，显示红点提示

**d. 系统设置**
- AI语气设置
- 是否公开名片动态
- 隐私设置
- **黑名单** - 屏蔽不想收到交换请求的人

**e. 底部**
- 会员权益介绍
- 联系我们 → 弹出表单收集建议

### 5.4 访客记录 (visitor)
- 展示访客列表（谁扫了我的二维码）
- 免费用户：只显示前3个基础信息（无联系方式）
- 付费用户：显示完整信息+名片详情

### 5.5 数据分析 (analytics)
- 访客数据详情
- 交换数据统计
- AI优化建议

### 5.6 AI功能 (aiFeatures)
- AI聊天助手对话框，可选模型（免费额度有限，部分付费）
- 来访者分析
- 联系人推荐
- 话术建议

### 5.7 管理 (management) - TabBar
- 账号设置
- 名片模板管理

## 八、技术可行性

1. **跳转微信**：如无法实现则暂时不做（"聊天"按钮可能去掉）
2. **分享到小红书**：微信小程序支持直接分享到微信，需要确认是否支持分享到小红书
3. **电话拨打**：wx.makePhoneCall
4. **气泡图/云图**：可用Canvas绘制，气泡位置随机分布，大小按数量比例
